**Hash Algorithms:**
- SHA-512 for primary integrity hashing
- SHA-256 for compatibility verification
- HMAC for authenticated sealing

**Timestamp System:**
- Device system time (primary)
- Network Time Protocol (NTP) verification when online
- Blockchain timestamp anchoring (optional)

**GPS Geolocation:**
- Precise coordinates (latitude/longitude)
- Accuracy measurement
- Altitude and bearing (if available)
- Location timestamp

**QR Code Integration:**
- Contains: Report ID, SHA-512 hash, timestamp, GPS
- Standard QR format for universal scanning
- Error correction level H (30% recovery)
- Embedded in report header and footer

**Digital Watermarking:**
- Invisible watermark in PDF
- Contains report hash and timestamp
- Survives printing and scanning
- Tamper-evident design
